== Description ==

Your WordPress menu to Responsive slide in navigation.

== Installation ==

1. Upload the `Mcl Slidein Nav` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to Settings > Mcl slidein Nav.
