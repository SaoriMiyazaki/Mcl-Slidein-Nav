== Description ==

Create from your WordPress menu Slide in animation nav.

Documentation & Demo: http://memocarilog.info/wordpress/8206

== Installation ==

1. Go to the Plugins Menu in WordPress.
2. Search for "Mcl Slidein Nav"
3. Click "Install".
